=============================
depth
=============================
板情報を取得します。

リクエスト
==============
/depth/{currency_pair}

currency_pairに指定できる値は :ref:`public-currency-pairs` を参照してください。


パラメータ
==============
なし

戻り値
==============
.. code-block:: python

    {
        "asks": [
            [
                134875.0,
                0.0063
            ],
            [
                134885.0,
                0.1639
            ],
            ...
        ],
        "bids": [
            [
                134870.0,
                0.01
            ],
            [
                134865.0,
                0.3066
            ],
            ...
        ]
    }


.. csv-table::
   :header: "キー", "詳細", "型"

   "asks", "売り板情報", "list"
   "bids", "買い板情報", "list"

補足
==============

取引種別
--------------

| 配列の最初が価格、最後が量
